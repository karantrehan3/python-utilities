Python Utilities API

PDF, image, and text processing — bundled into one FastAPI server. Unlock, compress, convert, hash, resize — one container, one port.

View on GitHub Explore endpoints

FastAPI · Dockerised · PyMuPDF · Pillow · Pydantic v2

What it does

📄

PDF tools

Unlock password-protected files, extract page ranges, combine images into a PDF, and compress with tunable image quality.

🖼️

Image tools

Resize, convert between formats, and inspect dimensions — via base64, URL, or direct file upload. Supports PNG, JPEG, WebP, GIF, BMP.

🔤

Text tools

Hash text with MD5, SHA-256, and more. Encode and decode with Base64 and hex. Lightweight and zero-config.

🐳

One container

All utilities share a single FastAPI process. One Docker image, one port, shared dependencies — no per-tool overhead.

📖

Auto-documented

Built-in Swagger UI and ReDoc. Every endpoint has typed request/response schemas generated from Pydantic models.

🧩

Easy to extend

Drop a new module into routes/api/v1/, wire it up in one line, and it inherits the full middleware stack.

Endpoints

PDF — /api/v1/pdf

EndpointMethodDescription
/unlockPOSTUnlock a password-protected PDF
/infoPOSTPage count, encryption status, metadata
/subsetPOSTExtract a page range into a new PDF
/from-imagesPOSTCombine images into a single PDF
/compressPOSTCompress a PDF (returns the file)
/compress/infoPOSTCompress a PDF (returns size-reduction stats)

Image — /api/v1/image

EndpointMethodDescription
/resizePOSTResize via base64 or URL
/convertPOSTConvert format via base64 or URL
/infoPOSTDimensions, format, size
/resize/filePOSTResize from file upload
/convert/filePOSTConvert from file upload
/info/filePOSTInfo from file upload
/formatsGETSupported image formats

Text — /api/v1/text

EndpointMethodDescription
/hashPOSTHash text (MD5, SHA-256, etc.)
/encodePOSTBase64 / hex encode
/decodePOSTBase64 / hex decode
/algorithmsGETSupported hash algorithms
/encodingsGETSupported encodings

Try it

# combine images into a PDF
curl -X POST http://localhost:4001/api/v1/pdf/from-images \
  -F "files=@page1.png" -F "files=@page2.jpg" \
  -o combined.pdf

# compress a PDF (tunable quality)
curl -X POST http://localhost:4001/api/v1/pdf/compress \
  -F "file=@large.pdf" -F "image_quality=60" \
  -o compressed.pdf

# hash some text
curl -X POST http://localhost:4001/api/v1/text/hash \
  -H "Content-Type: application/json" \
  -d '{"text": "hello", "algorithm": "sha256"}'

# resize an image
curl -X POST http://localhost:4001/api/v1/image/resize/file \
  -F "file=@photo.png" -F "width=200" -F "height=200"

Quick start

Clone the repo. git clone https://github.com/karantrehan3/python-utilities.git

Start with Docker. Run make start — builds the image and brings up the server on port 4001.

Or run locally. make install-dev && make dev-reload for uvicorn with hot-reload.

Open the docs. Visit localhost:4001/docs for the interactive Swagger UI.
