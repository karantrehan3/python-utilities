API Reference

Also a REST API

Every tool is an endpoint

The same backend that powers the UI is exposed as a standard REST API. Use curl, Postman, or any HTTP client. Full Swagger docs ship at /docs.

bash 
$ curl -X POST localhost:4001/api/v1/pdf/merge \
    -F "files=@doc1.pdf" -F "files=@doc2.pdf" \
    -o merged.pdf

✓ 200 OK — merged.pdf (340 KB, 8 pages)

Get Started

Quick Start

🐳 Docker

Build and start both services with one command.

make start

UI at localhost:4002 · API at localhost:4001

💻 Local Dev

Install dependencies and start with hot-reload.

make install && make dev

UI at localhost:3000 · API at localhost:4001
